# Example User Prompts

Below are some example natural language prompts that users can provide to the ValidateAI platform for dashboard validation workflows.

---


## 1.Validation for Argentina

```text
Can you validate the Tableau Overall dashboard data for all players from Argentina in the Preprod environment? I specifically want to compare the T Target Last12 metric between the UI dashboard and backend database records. Ignore any players where the team filter is not available, and make sure this is done using row-level comparison logic.
```
